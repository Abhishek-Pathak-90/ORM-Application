Orbit Response Matrix Analyzer

A powerful PyQt5 application for analyzing orbit response matrices from accelerator data

Developed by Abhishek Pathak

View on GitHub Get Started

Features

Dark-Themed GUI

Modern dark UI with multiple tabs for Correctors, BPMs, Response Matrices, and Error Matrices

Advanced Analysis

Time-domain and frequency-domain analysis with FFT-based spectra computation

Data Management

Import CSV data and manage device lists with automatic column identification

Export Options

Export tables as CSV and save plots as PNG/JPG formats

Installation

1. Clone the Repository

git clone https://github.com/Abhishek-Pathak-90/ORM-Application.git
cd ORM-Application

2. Install Dependencies

pip install pyqt5 matplotlib numpy pandas

3. Run the Application

python main.py

Documentation

Data Import

Load CSV files containing time-domain signals of BPMs and corrector magnets. The application automatically identifies valid columns.

Analysis

Compute FFT spectra, build orbit response matrices, and calculate measurement errors using partial derivative approach.

Visualization

View time-domain plots, FFT spectra, and heatmaps of response and error matrices with adjustable font sizes.
